HRA Accountant – Professional Website

A modern, single‑page business website for HRA Accountant, a leading Irish accounting firm. Built with HTML5, vanilla CSS (custom design system), and vanilla JavaScript – no heavy frameworks. Features a 3D hero tilt, particle background, scroll‑triggered animations, interactive pricing cards, and a full SPA routing system for service pages.

Tech Stack
- **HTML5** – semantic, accessible markup
- **CSS3** – custom properties, grid/flexbox, responsive design, animations
- **JavaScript (ES6)** – modular architecture, IntersectionObserver, Canvas API, hash‑based routing
- **External fonts:** Google Fonts (Sora, DM Serif Display)

Quick Start (Local Development)
1. **Clone or download** this repository to your machine.
2. Open the project folder in your code editor.
3. Launch `index.html` directly in a modern browser (no build step required).
4. To test service pages, click on any card or navigation link – they load dynamically via JavaScript.
5. To edit content, update the `pageContent` object inside `assets/js/main.js` or modify the HTML in `index.html`.
